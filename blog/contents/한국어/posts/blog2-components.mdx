# 제목

1. 사용법

```md
# 제목1

## 제목2

### 제목3
```

2. 컴포넌트

-   `H1`
-   `H2`
-   `H3`

`H3`이상으로 깊게 제목을 세분화 시키면 글의 구조가 다소 복잡해지는 단점이 존재합니다

<br />

3. `Table of content` 컴포넌트

-   `H1`과 자식 `H2`만 포함합니다
-   정상적인 `TOC` 이용을 위해, 제목의 글은 순수 텍스트만 포함 해야합니다

```md
🚫 # _Head_ **is** `head`
✅ # Head is head
```

## 사용법 - code block

1.  사용법

````md
```language
language code
```
````

````md
```js {1,2-3} showLineNumbers
const favorites = {
    animal: "sea-otter 🦦",
    fruits: ["🍎", "🍏", "🍓", "🍑", "🍍"],
}
```
````

코드 블럭 설정 옵션

1. 언어: `language` ( 필수 )
2. 강조할 줄: `{singleLineNumber, startLineNumber-endLineNumber}` ( 선택 )
3. 줄 수: `showLineNumbers` ( 선택 )

인수 제공: `language` `{singleLineNumber, startLineNumber-endLineNumber}` `showLineNumbers`

2.  결과

```js
const NAME = "NEXT_YOUR_HOME"
```

```js {1,2-3} showLineNumbers
const favorites = {
    animal: "sea-otter 🦦",
    fruits: ["🍎", "🍏", "🍓", "🍑", "🍍"],
}
```
