# 제목

1. 사용법

```md
# 제목1

## 제목2

### 제목3
```

2. 컴포넌트

-   `H1`
-   `H2`
-   `H3`

`H3`이상으로 깊게 제목을 세분화 시키면 글의 구조가 다소 복잡해지는 단점이 존재합니다

<br />

3. `Table of content` 컴포넌트

-   `H1`과 자식 `H2`만 포함합니다
-   정상적인 `TOC` 이용을 위해, 제목의 글은 순수 텍스트만 포함 해야합니다

```md
🚫 # _Head_ **is** `head`
✅ # Head is head
```

## code block 사용법

1.  사용법

````md
```language
language code
```
````

2. 코드 블럭 설정 옵션

    - 언어 ( _required_ ): `language`
    - 강조 ( _optional_ ): `{singleLineNumber, startLineNumber-endLineNumber}`
    - 줄 수 ( _optional_ ): `showLineNumber`

인수 제공: `language {lineNumber} showLineNumbers`

````md
```js {1,2-3} showLineNumbers
const favorites = {
    animal: "sea-otter 🦦",
    fruits: ["🍎", "🍏", "🍓", "🍑", "🍍"],
}
```
````

3.  결과

```js {1,2-3} showLineNumbers
const favorites = {
    animal: "sea-otter 🦦",
    fruits: ["🍎", "🍏", "🍓", "🍑", "🍍"],
}
```
