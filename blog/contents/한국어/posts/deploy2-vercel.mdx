# Vercel 계정 만들기

[계정 만들기](https://vercel.com/login), `Github`로 시작하는 것을 추천합니다!

---

# Let's building something new 페이지

자신의 `Github`로 가져온 **next your home repo** 선택 후, `import` 버튼 클릭

> repository가 한개도 안보여요!:question

1. [Github 설정](https://github.com/settings/installations)에 접속합니다
2. `vercel` Congure 버튼을 클릭합니다
3. 맨 하단의 `Repository access`옵션을 **_All repositories_**로 변경합니다

![import git repo](/Deploy/vercel/1.png)

---

# You're almost done 페이지

1. 프로젝트 이름 변경하고자 한다면 바꿉니다
2. `Deploy` 버튼을 클릭합니다
3. 자동으로 배포를 시작합니다

![configure project](/Deploy/vercel/2.png)

---

# Congratulations! 배포 완료 페이지

1. 위 과정을 거치면 'Github' repo를 불러온 후 자동으로 배포를 진행합니다
2. 아래 화면을 보신다면 축하드립니다🥳! 배포가 성공적으로 끝났습니다!

![deploy succesed!](/Deploy/vercel/3.png)

## 배포 링크 변경해보기

1. 상단 `settings` 클릭

![project setting](/Deploy/vercel/4.png)

2. 프로젝트 이름 변경

`project_name`을 변경하면

`https://vercel.com/{vercel_id}/{project_name}`

로 배포 링크가 변경됩니다

![change project name](/Deploy/vercel/5.png)
