# Google Search Console

## 속성 유형 선택

1. [사이트 접속하기](https://search.google.com/search-console/welcome)

2. **URL 접두어** 방식을 선택합니다

3. 배포 URL을 입력합니다

4. `계속`을 클릭합니다

![속성 유형 선택](/Deploy/search/1.png)

## 사이트 소유권 확인

1. `HTML` 방식을 선택합니다

2. 제시된 파일을 다운로드 합니다

![다운로드 방식](/Deploy/search/2.png)

3. `public`폴더에 해당 파일을 옮겨줍니다

4. 원격 저장소에 업데이트하기

-   원격 저장소에 `public`폴더에 저장한 파일을 `push`합니다
-   자동으로 재배포가 진행됩니다

## 저장된 파일로 소유권 확인

Google Search Console로 돌아온 후 진행하면, 확인 완료 매세지를 받습니다

이후 속성으로 이동을 클릭합니다

## Sitemap 등록

1. 왼쪽 색인의 `Sitemaps`를 클릭합니다

![sitemaps 클릭](/Deploy/search/3.png)

2. 새 사이트맵 추가란에 `sitemap.xml`을 입력합니다

> 왜 `sitemap.xml`을 입력하나요?:question

[public 폴더](https://nextjs.org/docs/basic-features/static-file-serving)에 `sitemap.xml`을 생성하기 때문입니다

![sitemap 등록하기](/Deploy/search/4.png)

3. 사이트맵이 성공적으로 제출되었는지 상태를 확인합니다

상태: `성공`

![sitemap 등록](/Deploy/search/5.png)

## 구글, 축하드립니다 🥳!

블로그가 **Google Search Console**에 정상적으로 등록되었습니다

# Naver Search Advisor

[Naver Search Advisor](https://searchadvisor.naver.com/console/board)에 접속합니다

## 사이트 등록

블로그 url을 입력합니다

![블로그 url](/Deploy/search/6.png)

## 사이트 소유권 확인 2

1. `HTML` 파일 업로드 방식을 선택합니다

![다운로드 방식](/Deploy/search/7.png)

2. `HTML 확인 파일`을 클릭하여 파일을 다운로드 합니다

3. 이후 **Google Search Console**과 동일한 방식으로 진행합니다

## 저장된 파일로 소유권 확인 2

재배포 후 `소유 확인` 버튼을 클릭합니다

확인 완료 매세지를 확인합니다

## Sitemap 등록 2

1. 왼쪽 요청의 `사이트맵 제출`을 클릭합니다

2. 새 사이트맵 추가란에 `sitemap.xml`을 입력합니다

![사이트맵 제출 클릭](/Deploy/search/9.png)

3. `확인`을 클릭하고 사이트맵이 성공적으로 제출되었는지 확인합니다

## RSS 등록

1. 왼쪽 요청의 `RSS 제출`을 클릭합니다

2. 새 RSS 추가란에 `rss.xml`을 입력합니다

![RSS 제출 클릭](/Deploy/search/10.png)

3. `확인`을 클릭하고 RSS가 성공적으로 제출되었는지 확인합니다

## 네이버, 축하드립니다 🥳!

블로그가 **Naver Search Advisor**에 정상적으로 등록되었습니다
