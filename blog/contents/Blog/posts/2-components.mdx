> `src/components/MDXAtoms` contains all components

# Title

1. How to use

```md
# Title 1

## Title 2

### Title 3
```

2. Components

-   `H1`
-   `H2`
-   `H3`

There is a disadvantage that the structure of the text becomes somewhat complicated if the title is deeply subdivided beyond `H3`

3. `Table of content` component

-   Only `H1` and `H2` are included
-   For normal use of `TOC`, the text of the title must contain only pure text

```md
🚫 # _Head_ **is** `head`
✅ # Head is head
```

## How to use code block

1. How to use

````md
```language
language code
```
````

2. Code Block Settings Options

    - Language ( _required_ ): `language`
    - Highlight ( _optional_ ): `{singleLineNumber, startLineNumber-endLineNumber}`
    - Number of lines ( _optional_ ): `showLineNumber`

Argument: `language {lineNumber} showLineNumbers`

````md
```js {1,2-3} showLineNumbers
const favorites = {
    animal: "sea-otter 🦦",
    fruits: ["🍎", "🍏", "🍓", "🍑", "🍍"],
}
```
````

3. Results

```js {1,2-3} showLineNumbers
const favorites = {
    animal: "sea-otter 🦦",
    fruits: ["🍎", "🍏", "🍓", "🍑", "🍍"],
}
```
